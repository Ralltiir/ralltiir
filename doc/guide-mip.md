# MIP接入
## 适用场景
适用于外部资源引入，并且资源引入方式必须是站长整体页面。
如果外部资源合作方式是"结构化"数据，则推荐使用**阿拉丁接入**。
使用iframe页面的形式接入[Superframe](http://sfe.baidu.com/sf/#about)。
