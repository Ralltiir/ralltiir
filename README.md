## 概述

Superframe是一个页面异步解决方案。

主要用于解决页面与页面的跳转体验糟糕的问题。

我们期望通过Superframe让Web的页面跳转也像Native App一样流畅。

了解更多信息，访问[Superframe Docs](https://searchfe.github.io/superframe)
